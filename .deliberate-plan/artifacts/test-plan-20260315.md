## Production Readiness Criteria

All tests pass = these guarantees hold:
1. **SSRF protection**: Only Supabase Storage URLs are fetched (no internal network access)
2. **Graceful degradation**: Any image failure falls back to text-only embedding (no user impact)
3. **Data integrity**: Embeddings are only overwritten when content actually changes (dedup)
4. **Race safety**: Concurrent updates can't produce double-embeds or lost writes
5. **PII safety**: No street addresses sent to external APIs
6. **Feature isolation**: Image embeddings can be disabled without affecting text search
7. **Model safety**: Cache entries from different models never cross-contaminate
8. **State machine correctness**: PENDING → PROCESSING → COMPLETED/PARTIAL/FAILED transitions are valid
