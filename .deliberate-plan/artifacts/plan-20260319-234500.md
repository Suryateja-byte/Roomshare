**File 2**: `src/app/api/messages/route.ts:76-78`

Before:
```typescript
export async function GET(request: Request) {
  try {
    const session = await auth();
```

After:
```typescript
export async function GET(request: Request) {
  try {
    // IP-based rate limit BEFORE auth() to prevent session-lookup flood
    const preAuthRl = await withRateLimit(request, {
      type: "messagesPreAuth",
      endpoint: "/api/messages:pre-auth",
    });
    if (preAuthRl) return preAuthRl;

    const session = await auth();
```

**Why 300/hr**: Per-user limit is 60/hr. IP-based must be 5x higher to avoid false positives on shared NATs/offices. 300/hr = 5 requests/min, generous for legitimate use.
