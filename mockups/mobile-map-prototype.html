5. Sprint 2: State Coordination (S2-1) - Flow Diagram

One-directional data flow via useReducer in MobileSearchContext.

  User Actions                    MobileSearchContext              UI Updates
  ============                    ===================              ==========

  Tap marker --------dispatch---> MARKER_TAP {id}
                                    snap: collapsed              -> Sheet collapses
                                    activeId: id                 -> Preview card shows
                                    focusSource: 'map'           -> Map centers on marker

  Drag sheet --------dispatch---> SHEET_DRAG {snap}
                                    snap: new value              -> Sheet animates
                                    activeId: null (if peek)     -> Preview hides

  Tap "List" pill ---dispatch---> TOGGLE_PRESS
                                    snap: peek (from collapsed)  -> Sheet expands to peek
                                       or collapsed (from peek)  -> Sheet collapses

  Tap map bg --------dispatch---> CLEAR_FOCUS
                                    activeId: null               -> Preview hides
                                    snap: unchanged              -> Sheet stays

  Apply filter ------dispatch---> FILTER_APPLY
                                    snap: peek                   -> Sheet shows results
                                                                 -> URL navigation fires

  Back button -------dispatch---> BACK_PRESS
                                    snap: snap - 1               -> Sheet steps down
                                    activeId: null (if needed)   -> Preview hides
