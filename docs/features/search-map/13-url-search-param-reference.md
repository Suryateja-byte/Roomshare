# URL / Search-Param Reference

This file is the reconstruction reference for `/search` URL state. It documents current behavior only. The source of truth is the parser in `src/lib/search-params.ts` and the URL helper in `src/lib/search/search-query.ts`; this page does not define future intended behavior. Evidence: `evidence-register.md` C003, C021, C039, C055.

## Canonical Model

`parseSearchParams` turns raw URL params into normalized filters, sort, requested page, bounds flags, and browse-mode flags. `normalizeSearchQuery` and `serializeSearchQuery` then provide a client-facing canonical URL model. Evidence: `src/lib/search-params.ts`:L1-L4, L790-L909; `src/lib/search/search-query.ts`:L152-L279.

Important rules:

- Canonical serialized URLs use canonical names such as `locationLabel`, `moveInDate`, `minPrice`, `maxPrice`, `minSlots`, `page`, and `cursor`.
- Legacy aliases are accepted inbound but are not serialized back out as aliases. Evidence: `src/lib/search-params.ts`:L49-L82; `src/lib/search/search-query.ts`:L211-L279.
- Repeated and comma-separated array params are preserved as arrays, normalized, deduped, and sorted during client URL serialization. Evidence: `src/lib/search-params.ts`:L226-L253, L302-L318; `src/lib/search/search-query.ts`:L97-L99, L222-L236.
- Result-affecting filter keys are tracked separately from pagination, sort, and map viewport keys. Evidence: `src/lib/search-params.ts`:L203-L224.
- `location`, `filter`, `sort`, `map-pan`, and `saved-search-reopen` query changes clear `page` and `cursor`. Evidence: `src/lib/search/search-query.ts`:L317-L375; `src/__tests__/lib/search/search-query.test.ts`:L48-L72.

## Canonical Serialization Notes

`serializeSearchQuery` writes only normalized, canonical values. It omits `sort=recommended`, omits `page=1`, prefers `cursor` over `page`, writes `minSlots` instead of `minAvailableSlots`, writes `locationLabel` instead of `where`, and rounds explicit bounds to 3 decimals. Evidence: `src/lib/search/search-query.ts`:L203-L279; `src/__tests__/lib/search/search-query.test.ts`:L10-L46, L119-L146, L178-L210.
